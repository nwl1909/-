# ♠ Дурак онлайн

Классическая карточная игра «Дурак» (подкидной, 36 карт) — прямо в браузере, без установки и без сервера. Один HTML-файл, ванильный JS, ноль зависимостей для сборки.

**[Играть →](https://YOUR_USERNAME.github.io/YOUR_REPO/)** *(ссылка появится после публикации через GitHub Pages, см. ниже)*

## Как запустить локально

Просто откройте `index.html` в браузере — двойным кликом или через `open index.html` / `start index.html`.

Либо через локальный сервер (не обязательно, но иногда удобнее):

```bash
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

## Как опубликовать на GitHub Pages

1. Создайте репозиторий на GitHub и загрузите в него `index.html` (и этот `README.md`).
2. Зайдите в **Settings → Pages**.
3. В разделе **Build and deployment** выберите **Source: Deploy from a branch**.
4. Укажите branch `main` (или `master`) и папку `/ (root)`, нажмите **Save**.
5. Через минуту-две сайт будет доступен по адресу `https://<ваш_логин>.github.io/<название_репозитория>/`.

Файл не требует сборки, бандлера или npm — GitHub Pages отдаёт `index.html` как есть.

## Правила игры

- Колода — 36 карт (от 6 до туза), козырь определяется случайной картой при раздаче.
- Каждому — по 6 карт. Ходит первым тот, у кого младший козырь.
- Атакующий кладёт карту, защищающийся бьёт её старшей картой той же масти или козырем.
- Пока защищающийся не отбился, можно подкидывать карты того же достоинства, что уже на столе (не больше карт, чем осталось у защищающегося на руках).
- Отбился — карты уходят в «бито», ход атаки переходит к защищавшемуся.
- Не отбился — забирает все карты со стола себе, атака переходит к следующему.
- После розыгрыша игроки добирают карты из колоды до 6, начиная с атаковавшего.
- Цель — избавиться от всех карт первым. Оставшийся с картами, когда колода и рука соперника пусты — «дурак».

Полное описание правил доступно прямо в игре по кнопке **«?»**.

## Технологии

- Чистый HTML/CSS/JS — без React, без сборки, без зависимостей
- Один файл `index.html`, полностью адаптивный (мобильные и десктоп)
- Шрифты: Playfair Display + Inter (Google Fonts)
- Игра против простого ИИ-соперника (бьёт и ходит наименьшими картами, иногда меняет стратегию для непредсказуемости)
